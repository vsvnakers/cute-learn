# 第七课：MMA 可视化演示

## 7.1 MMA 操作可视化

### 16×8×8 MMA 数据流

```
输入 A (16×8)              输入 B (8×8)               输出 D (16×8)
┌────────────────┐        ┌────────────────┐        ┌────────────────┐
│ A00  A01  ...  │        │ B00  B01  ...  │        │ D00  D01  ...  │
│ A10  A11  ...  │        │ B10  B11  ...  │        │ D10  D11  ...  │
│ ...            │   ×    │ ...            │   =    │ ...            │
│ A15,0 ... A15,7│        │ B70  B71  ... B77│      │ D15,0 ... D15,7│
└────────────────┘        └────────────────┘        └────────────────┘

计算过程 (单个输出元素):
D[i][j] = Σ(k=0 to 7) A[i][k] × B[k][j] + C[i][j]
```

## 7.2 寄存器布局可视化

### FP16 MMA 寄存器分配

```
m16n8k8 FP16 MMA 的寄存器需求:

A 矩阵 (16×8 FP16 = 128 元素):
┌─────────────────────────────────────────────────────┐
│  每线程 4 个 FP16 = 1 个 32-bit 寄存器               │
│  32 线程 × 4 FP16 = 128 FP16 (完整 A 矩阵)          │
│                                                      │
│  Thread 0:  A[0:1, 0:1]  →  a0 = {A00, A01, A10, A11}│
│  Thread 1:  A[0:1, 2:3]  →  a1 = {A02, A03, A12, A13}│
│  ...                                                 │
└─────────────────────────────────────────────────────┘

B 矩阵 (8×8 FP16 = 64 元素):
┌─────────────────────────────────────────────────────┐
│  每线程 2 个 FP16 = 0.5 个 32-bit 寄存器            │
│  32 线程 × 2 FP16 = 64 FP16 (完整 B 矩阵)           │
│                                                      │
│  Thread 0:  B[0:1, 0]  →  b0 = {B00, B10}           │
│  Thread 1:  B[0:1, 1]  →  b1 = {B01, B11}           │
│  ...                                                 │
└─────────────────────────────────────────────────────┘

C/D 矩阵 (16×8 FP16 = 128 元素):
┌─────────────────────────────────────────────────────┐
│  每线程 4 个 FP16 = 1 个 32-bit 寄存器              │
│  32 线程 × 4 FP16 = 128 FP16 (完整 C/D 矩阵)        │
│                                                      │
│  Thread 0:  D[0:1, 0:1]  →  d0 = {D00, D01, D10, D11}│
│  Thread 1:  D[0:1, 2:3]  →  d1 = {D02, D03, D12, D13}│
│  ...                                                 │
└─────────────────────────────────────────────────────┘
```

### FP32 累加器布局

```
FP32 累加器 (用于 FP16 输入，FP32 输出):

每个线程 4 个 FP32 寄存器 = 16 个 FP16 精度:
┌────────────────────────────────────────┐
│  reg_d[0] = D[0, 0] (FP32)             │
│  reg_d[1] = D[0, 1] (FP32)             │
│  reg_d[2] = D[1, 0] (FP32)             │
│  reg_d[3] = D[1, 1] (FP32)             │
└────────────────────────────────────────┘

32 线程 × 4 FP32 = 128 FP32 (完整输出)
```

## 7.3 MMA 流水线可视化

### 双缓冲流水线

```
时间 →

共享内存 Bank:
┌─────────────┬─────────────┬─────────────┬─────────────┐
│   Bank 0    │   Bank 1    │   Bank 0    │   Bank 1    │
│   (Load 0)  │   (Load 1)  │   (Load 2)  │   (Load 3)  │
├─────────────┼─────────────┼─────────────┼─────────────┤
│   Compute 0 │   Compute 1 │   Compute 2 │   Compute 3 │
└─────────────┴─────────────┴─────────────┴─────────────┘

时序图:
Cycle 0:  Load A0, B0 → Bank 0
Cycle 1:  Wait → Compute Bank 0
Cycle 2:  Load A1, B1 → Bank 1
Cycle 3:  Wait → Compute Bank 1
Cycle 4:  Load A2, B2 → Bank 0
Cycle 5:  Wait → Compute Bank 0
...

优势:
- 加载和计算重叠
- 隐藏内存延迟
- 提高吞吐量
```
