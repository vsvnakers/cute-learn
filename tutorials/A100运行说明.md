# A100 运行说明

## 快速开始

### 1. 上传到 A100

```bash
# 方法 1: scp 上传
scp cute_tutorials.tar.gz user@a100-server:~/

# 方法 2: rsync 上传
rsync -av cute_tutorials.tar.gz user@a100-server:~/
```

### 2. 在 A100 上解压编译

```bash
# SSH 登录
ssh user@a100-server

# 解压
cd ~
tar -xzf cute_tutorials.tar.gz
cd tutorials

# 一键编译运行
bash run_on_a100.sh
```

### 3. 手动运行单个教程

```bash
cd ~/tutorials/build

# 基础教程
./01_layout_basic
./02_swizzle_basic
./03_bank_conflict
./04_mma_basic
./05_gemm

# Attention
./06_flash_attention
./13_attention_v2

# 性能测试
./14_benchmark
```

## 系统要求

- **GPU**: NVIDIA A100 (SM80)
- **CUDA**: 11.0 或更高版本
- **CMake**: 3.18 或更高版本
- **编译器**: g++ 7+ 或 nvcc

## 检查环境

```bash
# 检查 GPU
nvidia-smi

# 检查 CUDA 版本
nvcc --version

# 检查 CMake
cmake --version
```

## 常见问题

### Q: 编译失败 "no cuda-cpp compiler"
**A**: 安装 CUDA Toolkit 11.0+

### Q: 运行时 "CUDA error: no CUDA-capable device"
**A**: 确认 A100 驱动已正确安装

### Q: "Permission denied"
**A**: `chmod +x run_on_a100.sh`
