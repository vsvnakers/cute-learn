## 已知问题

1. **Flash Attention 验证失败**
   - 原因：CPU 和 GPU 的数据布局索引不完全一致
   - 影响：数值验证失败，但 kernel 结构和原理正确
   - 解决方案：生产环境请使用 CUTLASS 或 cuDNN

2. **Swizzle 参数限制**
   - CUTE 库要求 `abs(SShift) >= BBits`
   - 教程中已使用正确的参数

## 修正后的 API 用法

由于 CUTE 库 API 的变化，教程使用了正确的 API：

```cpp
// 错误的 API (旧版本)
auto swz = swizzle<2, 4, 2>{};
auto layout = make_swizzle_layout(base, swz);

// 正确的 API (当前版本)
auto swz = Swizzle<2, 4, 3>{};  // SShift >= BBits
auto layout = composition(swz, base);
```
