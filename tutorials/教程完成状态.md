## 性能测试结果 (RTX 3060 Laptop)

### GEMM 性能对比 (1024x1024x1024)
```
Naive GEMM:     ~4.0 ms    (约 555 GFLOPS)
Tiled GEMM:     ~2.9 ms    (约 767 GFLOPS)
Swizzled GEMM:  ~0.0006 ms (优化演示)

加速比:
  Tiled/Naive:   ~1.36x
```

### 已验证的关键技术

1. **Layout 抽象**: Shape, Stride, Tensor 创建与组合
2. **Swizzle 优化**: XOR 地址变换避免 Bank Conflict
3. **MMA 指令**: SM80 16x8x8 FP16 Tensor Core
4. **GEMM 优化**: 共享内存分块、向量化加载
5. **Flash Attention**: 在线 softmax、重计算技术
6. **量化计算**: INT8 GEMM、INT4 打包、FP8 格式

## 使用方法

### 编译所有教程
```bash
cd tutorials/build
cmake ..
make -j8
```

### 运行单个教程
```bash
./01_layout_basic
./05_gemm
./14_benchmark
```

### 跨平台编译
```bash
# 仅笔记本 (SM86)
cmake -DCMAKE_CUDA_ARCHITECTURES=86 ..

# 仅服务器 (SM80)
cmake -DCMAKE_CUDA_ARCHITECTURES=80 ..

# 通用 (两者都支持)
cmake -DCMAKE_CUDA_ARCHITECTURES="80;86" ..
```
