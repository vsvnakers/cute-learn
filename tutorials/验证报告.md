## A100 (SM80) 兼容性说明

### 为什么能在 A100 上运行？

1. **编译时包含双架构**: `set(CMAKE_CUDA_ARCHITECTURES 80 86)`
   - 生成的 fatbin 同时包含 sm_80 和 sm_86 的机器码

2. **CUDA 运行时选择**:
   - 程序启动时，CUDA 驱动检测 GPU 型号
   - 自动加载匹配的架构代码 (A100 → sm_80, RTX 3060 → sm_86)

3. **向后兼容**:
   - sm_80 代码可以在任何 SM80+ 的 GPU 上运行
   - sm_86 代码可以在任何 SM86+ 的 GPU 上运行

### 在 A100 上验证

```bash
# SSH 连接到 A100 服务器后
cd /path/to/tutorials/build

# 确认 GPU 型号
nvidia-smi

# 运行测试
./05_gemm
./13_attention_v2
./14_benchmark
```
