# 第五课：完整 GEMM 实现

## 1. GEMM 基础

GEMM (General Matrix Multiply) 是深度学习中最核心的运算：

```
C = A × B

其中:
- A: M×K 矩阵
- B: K×N 矩阵
- C: M×N 矩阵
```

## 2. GEMM 优化层次

### 层次 1：基础实现
- 简单的三层循环
- 效率：< 1%

### 层次 2：共享内存分块
- 使用共享内存减少全局内存访问
- 效率：5-10%

### 层次 3：Tensor Core
- 使用 MMA 指令
- 效率：30-50%

### 层次 4：高级优化
- 流水线重叠
- 向量化的内存访问
- Warp 级优化
- 效率：60-80%

## 3. 分块策略

```
全局内存 (M×N)
    ↓
分块加载到共享内存 (Tile_M × Tile_K)
    ↓
Tensor Core MMA (16×8×8)
    ↓
累加到寄存器
```
